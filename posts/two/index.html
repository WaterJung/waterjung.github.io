介绍

Hugo是一个用Go编写的静态站点生成器，由于具有丰富的主题资源和惊人的生成速度而备受青睐。本博客即是基于Hugo搭建

安装Hugo

方式一：

  • hugo是托管在GitHub上的项目，我们可以直接到到这里下载安装 根据自己的系统选择合适的hugo版本，我的是Win10 64位，所以选择下图版本

PS:hugo版本随时间更新，不一定为此篇文章的版本，大家可以选择当前最新版本进行下载

下载后对压缩包进行解压得到hugo.exe，并把它配置到环境变量中。如果输入命令出现下图则说明可以使用hugo了。

hugo version

方式二：

  • 如果你是macOS用户，请使用Homebrew快速安装
brew install hugo
  • 如果你是Debian或Ubuntu用户，请使用apt快速安装
sudo apt-get install hugo
  • 如果你是Windows用户，可以使用Chocolatey或者Scoop快速安装，取决于你使用什么包管理
choco install hugo -confirm

scoop install hugo

生成本地站点

hugo new site 路径/站点名称

hugo new site 是新建一个网站的命令，后添加路径和站点名称；在当前路径下生成站点，路径可以省略。

  • 创建完成后，在 blog 文件夹下会生成以下文件结构：
.
├── archetypes # 存放生成博客的模版
├── assets # 存放被 Hugo Pipes 处理的文件
├── config # 存放 hugo 配置文件 支持 JSON YAML TOML 三种格式配置文件
├── content # 存放 markdown 文件
├── data # 存放 Hugo 处理的数据
├── layouts # 存放布局文件
├── static # 存放静态文件 图片 CSS JS文件
└── themes # 存放主题
  • 这样就快速地生成了一个空白的站点，这个时候如果你输入 hugo server 的命令（运行Hugo），然后在浏览器里输入 localhost:1313 会发现是空白的，主要是因为里面什么东西都没有，需要我们自己下载主题。

下载皮肤主题

  • 在Hugo主题里面挑选自己喜欢的主题，跳转到相关的 github 页面，clone 到自己的 blog/themes 文件夹当中。
cd myblog
cd themes
git clone https://github.com/g1eny0ung/hugo-theme-dream.git

站点配置

  • 进入 myblog/themes/hugo-theme-dream/exampleSite 文件夹，将 config.tom 文件、/hugo-theme-dream下的 static和i18n文件一同拷贝到根目录下

打开此配置文件

baseurl = "https://example.com"  
languageCode = "en" //语言  中文则选择输入zh
defaultContentLanguage = "en"
title = "Hugo Theme Dream" // 博客的标题
theme = "hugo-theme-dream" //主题

# copyright = "" //版权

# googleAnalytics = "" //网站分析，我们用不到，默认注释掉

# disqusShortname = ""  

# enableRobotsTXT = true

[params]
  #background = "black"  //背景颜色
  backgroundImage = "/img/background.jpg" //背景图片
  linkColor = "seagreen"

  author = "Yue Yang" //作者
  # description = "" //描述
  avatar = "/img/avatar.jpg" //头像
  motto = "Motto" //
  
  email = "g1enyy0ung@gmail.com"
  # github = ""
  # linkedin = ""
  # codepen = ""
  # stackoverflow = ""

  siteStartYear = 2016 //年份起始

  # favicon = "/favicon.ico"

  # dark mode
  darkLinkColor = "darkseagreen"
  darkNav = true 
  dark404Button = true

ps:背景和背景图片二选一，背景图片放在/static/img 下

写一篇文章

新建一篇文章比较简单，命令如下：

hugo new site posts/xxx.md

ps：这个命令会使用模板创建文件，首先查找用户的模板文件，没有就会查找主题的模板文件。

文章的格式为markdown，很好上手，大家自行去探索。

启动 Hugo

命令行输入以下命令，启动 Hugo :

hugo server

效果如下图：

打包

  • 为了部署到线上，需要将 Markdown 文件打包成 HTML 文件。打包命令如下，hugo-theme-dream 是主题名：
hugo -t hugo-theme-dream

此时在根目录下多了一个public文件夹

部署到 Github Pages

  • 打包之后就是纯 HTML 文件，理论上所有支持部署静态页面的网站都是支持的。
  • 详情请访问https://gohugo.io/hosting-and-deployment/

前提是去github创建一个以github用户名小写.github.io的仓库

下面这些是git命令
cd public 

git init 

git add . 

git commit -m "我的 hugo 博客第一次提交"

git remote add origin https://github.com/github用户名/github用户名全小写.github.io.git

git push -u origin master  输入用户名及密码
